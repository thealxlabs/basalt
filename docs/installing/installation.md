### Installation
To fully experience Basalt, complete the following steps.
1. Install all Basalt files (you can do this through our [website](https://sites.google.com/view/basaltlang/home))
2. Copy the path to the `Basalt` folder.
3. Open **Environment Variables** in Windows.
4. Under "User variables", find **Path**, click **Edit**, then **New**.
5. Paste the path to your Basalt folder.
6. Restart your terminal.

Now you can run `basalt` from anywhere

### How to run programs
Once Basalt is added to your environment variables, you can run any `.b` file from your terminal using the following command:

```basalt path/to/yourfile.b```

### Test Basalt
After installing, you are encouraged test Basalt. Within the folder use the command:

```Test_Basalt```

This ensures that all the files are properly installed and that your basalt is working as it should.
